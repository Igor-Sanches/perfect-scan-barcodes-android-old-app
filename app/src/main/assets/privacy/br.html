Política de Privacidade

Perfect Scan não coleta, armazenar ou salvar nenhuma informação pessoal do usuário, nem para o uso da aplicação. O aplicativo somente acessa informações como câmera do dispositivo, agenda de contatos, memória removível (SD Card), aplicativo de mensagens para compartilhar e impressoras.

Note que o aplicativo não acessar ou transmite informações pela internet, e os navegadores que estão disponíveis para executar links não de responsabilidades dos próprios desenvolvedores, e só adicionamos os navegadores liberados pelos desenvolvedores do mesmo, todos os dados do aplicativo são armazenados em um local seguro e os downloads são armazenados em um pasta temporária para garantir a segurança e não transmitir virus no aparelho.

As informações pessoais que o aplicativo acessar são eles: *Nome do fabricante do aparelho, modelo, build, linguagem e local do app. Mais essas informações so vem a nos por meio do próprio usuário com a função ¨Traduzir¨ que precisamos das informações para garantir um bom funcionamento e a qual idioma sera a tradução, e também note que estas funções são enviados pelo usuário via email e ao mandar e abrir o aplicativo você esta aceitando estes termos.

Marcas

ZXing, QR Code, Data Matrix, Aztec, Code 128, Code 39, Code 93, Code EAN 13, Code EAN 9, Code ITF, Code MSI, Code PDF 417, Code PLESSEY, Code UPC A e Code UPC E são marcas comerciais de seus respectivos proprietários.
